# Acceptance Plane™ Public Readiness Kit v0.2.0

**자율적 AI action이 impact를 만들기 전, 조직이 어떤 evidence를 물어야 하는지 토론하기 위한 public readiness kit입니다.**  
**제품 구현체가 아닙니다. 표준이 아닙니다. 특허 라이선스가 아닙니다. 인증 권리를 주지 않습니다.**

**Not a product implementation. Not a standard. No patent license. No certification right.**

v0.2.0은 기존 5-minute scenario-card linter 위에 다음을 추가합니다.

- Action Acceptance Readiness Index
- NIST AI RMF / OWASP agentic-AI / Five Eyes-CISA / ISO/IEC 42001 public-safe crosswalk
- Public Evidence Ladder
- Board-to-builder workshop pack
- Procurement due-diligence pack
- Separate conformance artifact pointer lock
- Release artifact verification and stronger boundary QA

핵심 질문은 하나입니다.

> 이 정확한 autonomous action이 지금 이 protected system에 의해 accept되어도 되는가?

이 kit은 교육, readiness discussion, procurement alignment, tabletop review, category literacy를 위한 것입니다. production verifier logic, cryptographic binding method, exact evidence schema, non-bypassable enforcement mechanism, conformance vector, patent claim chart, implementation license를 포함하지 않습니다.
