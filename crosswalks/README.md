# Standards and Security Crosswalks

Boundary: educational crosswalks only. These files are not compliance mappings, legal opinions, certification controls, audit workpapers, or formal standards alignments.

Purpose:

> When an organization says “we follow NIST, OWASP, Five Eyes/CISA guidance, or ISO/IEC 42001,” this kit helps them ask the missing action-level question: what evidence must be present before an autonomous action creates impact?

Run:

```bash
make crosswalk-check
```
