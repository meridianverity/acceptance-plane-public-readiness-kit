# Scenario Taxonomy

Boundary: coverage for public education only. This is not conformance coverage, certification coverage, compliance coverage, or implementation test coverage.

The v0.2.0 public corpus contains 100 scenario cards across ten domains. Each card has a discussion decision label: ACCEPT, HOLD, or REFUSE. The label is a tabletop discussion label, not an operational allow/deny result.

Evidence category family used in the public corpus:

- action
- actor
- delegation
- scope
- target
- runtime_or_environment
- policy_state
- freshness
- revocation
- replay_or_duplicate
- consequence
- decision_reason
