# Publication Boundary

## Publish

- Category definition.
- Public workflow diagrams.
- Vocabulary.
- Scenario corpus.
- RFP questions.
- Human-impact and public-interest guidance.
- Non-production educational examples.
- Public readiness checklists.

## Protect

- Exact API schemas.
- Full evidence object models.
- Cryptographic binding methods.
- Non-bypassable enforcement pipelines.
- PermitReceipt implementation details.
- Signed test vector corpora.
- Conformance certificate registry designs.
- Hardware-specific adapters.
- Claim charts and partner-specific mappings.
- Unpublished patent language.

## Reason

The goal is to create a durable public category while reducing unnecessary implementation disclosure.
