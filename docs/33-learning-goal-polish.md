# Learning-goal polish — v0.3.3

v0.3.3 closed a visible scenario-language issue found after v0.3.2: ACCEPT-card learning goals that used a repeated reaches-before pattern.

The patch rewrites ACCEPT learning goals so they describe evidence checking in plain public language:

- current evidence
- scoped evidence
- target-matched evidence
- action impact boundary
- people named in the scenario

This is editorial stewardship only. It does not add verifier logic, cryptographic evidence, conformance vectors, certification rights, production allow/deny behavior, patent claim charts, or implementation licenses.
