# Facilitator reproduction path

The v0.3.4 facilitator path helps third parties reproduce the public education workflow without becoming certifiers or implementers.

Minimum steps:

1. Verify the release ZIP and SHA sidecar.
2. Run `make qa-full`.
3. Generate a public readiness report.
4. Use one workshop flow.
5. Fill the public result template.

Boundary: facilitator reproduction is educational reproducibility only. It is not certification, compliance approval, procurement approval, production readiness, conformance approval, implementation validation, legal advice, or patent licensing.
