# Procurement Readiness Rubric

This rubric helps buyers evaluate whether vendors understand action acceptance. It is not a certification.

| Score | Public meaning |
| --- | --- |
| 0 | Vendor treats autonomous actions as ordinary access-control events. |
| 1 | Vendor can list high-consequence action types. |
| 2 | Vendor can identify acceptance boundaries. |
| 3 | Vendor can describe evidence categories at the gate. |
| 4 | Vendor can explain ACCEPT / HOLD / REFUSE behavior. |
| 5 | Vendor can demonstrate production-grade, properly authorized enforcement with verifier-ready evidence and fail-closed behavior. |

## Required buyer caution

A public readiness score is not proof of product safety. Production enforcement should be evaluated through appropriate authorization, legal review, security review, domain validation, and independent audit.
