# AAIF Feedback / Working-Group-Fit Inquiry

This document is for outreach discipline. It is not a hosted-project proposal.

## Position

Meridian Verity Group is releasing the Acceptance Plane™ Public Readiness Kit as a free public readiness kit with a 5-minute educational scenario-card linter for developers.

We are not representing this repository as:

- an AAIF hosted project;
- a submitted project donation;
- a formal standard;
- a reference implementation;
- a product implementation;
- a conformance suite;
- a certification kit;
- a patent license;
- a production deployment artifact.
