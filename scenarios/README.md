# Acceptance Scenario Corpus

This directory contains shared metadata for the 100 public scenario cards used for teaching, tabletop exercises, procurement conversations, and readiness planning.

Scenario card JSON files live in two upload-friendly root directories:

- `scenarios_batch_1/` contains `AP-SCEN-001.json` through `AP-SCEN-050.json`.
- `scenarios_batch_2/` contains `AP-SCEN-051.json` through `AP-SCEN-100.json`.

The index, schema, and format guide remain in this `scenarios/` metadata directory.

The cards are not conformance vectors, certification artifacts, implementation disclosures, or production tests.

## Decisions

- ACCEPT: evidence is current, scoped, consistent, and sufficient.
- HOLD: proof is incomplete, stale, ambiguous, high-risk, or requires step-up review.
- REFUSE: proof fails, authority is revoked, target is mismatched, request is replayed, or action is unsafe/outside scope.
