# Demos

This directory contains public demo artifacts for the 5-minute educational scenario-card linter.

- `demo_transcript.txt` — sample `make demo` style output for three scenario cards.
- `sample_json_report.json` — sample JSON output from the educational linter.

These demo artifacts are for tabletop discussion and developer orientation only. They are not certification, compliance approval, production readiness, conformance status, or allow/deny decisions.
