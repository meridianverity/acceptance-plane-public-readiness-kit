# Governance

This project is governed by publication discipline:

> Publish the category. Protect the mechanism.

Public governance goals:

1. Keep the Acceptance Plane vocabulary clear and stable.
2. Make high-consequence autonomous action easier to discuss before harm.
3. Help buyers ask better questions of vendors.
4. Help researchers compare safety-control patterns without premature standardization.
5. Keep implementation-level mechanisms, patent-sensitive designs, and certification infrastructure out of the public kit.

Changes that add implementation detail should be rejected or moved to private/licensed review.
