# Action Acceptance Readiness Index — Scoring Model

Boundary: educational readiness signal only. Not certification, compliance approval, procurement approval, production readiness, or an allow/deny decision.

Each item is scored:

- YES = 2 points
- PARTIAL = 1 point
- NO = 0 points
- UNKNOWN = 0 points

Discussion maturity is `earned_points / maximum_points * 100`.

Posture bands:

- `FOUNDATIONAL_DISCUSSION_NEEDED`: below 60
- `HOLD_FOR_IMPLEMENTATION_REVIEW`: 60 to 79
- `DISCUSSION_READY_FOR_CONFORMANCE_HANDOFF`: 80 or above

The posture names are intentionally discussion-oriented. They do not approve deployment, certify a vendor, establish compliance, or replace separate conformance artifacts.
