# Vendor Response Red Flags

- Claims that agent access alone is enough for action acceptance.
- No freshness or revocation model.
- No replay or duplicate-action discussion.
- No human escalation path for HOLD.
- No distinction between permission and proof.
- No boundary between tool execution and external impact.
- No independent verification path when conformance evidence is requested.
- Unsupported claims of approval, certification, or compliance based only on public readiness material.

Boundary: red flags for discussion only; not vendor disqualification criteria.
