# Dependencies

## Runtime

The 5-minute educational scenario-card linter uses the Python standard library only.

No runtime package installation is required.

## Tested command path

```bash
python3 tools/scenario-card-lint/scenario_card_lint.py --summary scenarios_batch_1/AP-SCEN-001.json scenarios_batch_1/AP-SCEN-050.json scenarios_batch_2/AP-SCEN-100.json
```

## External services

None.

The kit does not call external APIs, does not require cloud credentials, does not connect to a registry, and does not perform production verification.
