# Public Procurement Scorecard

This scorecard is educational. It is not certification, compliance approval, conformance status, or a substitute for legal, security, clinical, financial, safety, or domain review.

| Area | 0 | 1 | 2 | 3 |
|---|---:|---:|---:|---:|
| Action inventory | no inventory | partial list | high-risk actions mapped | action classes tied to owners |
| Acceptance boundary | not identified | informally known | documented per high-risk action | reviewed before deployment |
| Evidence categories | logs only | identity and access | action/scope/target/freshness mapped | gaps tracked and remediated |
| HOLD path | none | manual exception | defined step-up review | monitored and reported |
| REFUSE path | ad hoc | policy exists | fail-closed posture for major gaps | tested in tabletop scenarios |
| Receipt readiness | vague logs | structured logs | verifier-ready categories | independent review path planned |
| Vendor posture | claims access is enough | partial answer | clear acceptance language | supports production evaluation path |

## Scoring guidance

- 0–6: not ready for high-consequence autonomy.
- 7–12: boundary awareness emerging.
- 13–18: public readiness developing.
- 19–21: ready to evaluate authorized production mechanisms.

Do not call any score “certified.”
