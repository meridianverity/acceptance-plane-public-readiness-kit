# Public adoption evidence template

Use this template to record public, non-confidential adoption evidence for the readiness kit.

- Organization type: board / buyer / builder / public-sector / nonprofit / research / other
- Public workflow used: scenario-card linter / readiness index / workshop packet / procurement questions / browser demo
- Public artifacts generated: readiness report / decision log / RFP question set / facilitator notes
- Private material excluded: implementation details, exact evidence schemas, cryptographic mechanisms, production deployment maps, partner/customer architecture, claim charts, legal analysis, and confidential procurement records

Boundary: public adoption evidence only. Not certification, compliance approval, procurement approval, production readiness, conformance evidence, implementation validation, endorsement, or patent license.
