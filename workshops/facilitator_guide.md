# Facilitator Guide

Use this guide to run a public-safe Acceptance Plane readiness workshop.

Facilitator rules:

- Keep the conversation at the action-before-impact boundary.
- Do not request private implementation mechanisms, patent claim details, customer architecture, or production keys.
- Treat scenario decisions as discussion labels only.
- Use HOLD whenever evidence is incomplete, stale, ambiguous, or outside scope.
- Close with one owner for the next readiness step.

Run `make workshop-demo` to generate a sample packet in `/tmp/acceptance-plane-workshop-pack/`.
