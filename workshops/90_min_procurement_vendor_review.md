# 90-Minute Procurement / Vendor Review

Goal: help a buyer ask action-level evidence questions without requesting private implementation mechanisms.

Agenda:

1. Identify high-consequence agentic workflows.
2. Select five scenario cards relevant to the vendor category.
3. Use `procurement/vendor_disclosure_questions.md`.
4. Review `procurement/vendor_response_red_flags.md`.
5. Draft RFP language from `procurement/rfp_clause_library.md`.

Boundary: procurement readiness discussion only; not vendor approval or certification.
