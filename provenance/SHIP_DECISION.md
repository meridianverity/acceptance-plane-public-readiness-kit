## Decision

Ship for public GitHub release.

## Rationale

The release passes the public-readiness gates:

- 100 public scenario cards.
- Two upload-friendly scenario batch directories with 50 cards each.
- Balanced ACCEPT / HOLD / REFUSE discussion labels.
- Ten-domain scenario coverage.
- Full-repo public boundary scan with no unsafe hits.
- Strict SHA-256 manifest verification.
- Clean checkout compatibility.
- Developer-first 5-minute demo.
- Split-license notice.
- AAIF posture framed as feedback / working-group-fit inquiry, not hosted-project submission.
- Public release guide uses a standard tag and `--verify-tag`.

## Boundary

This is a free public readiness kit with a 5-minute educational scenario-card linter for developers.

It is not a product implementation, formal standard, reference implementation, conformance suite, certification kit, patent license, production verifier, production deployment artifact, procurement approval, compliance approval, or allow/deny system.
